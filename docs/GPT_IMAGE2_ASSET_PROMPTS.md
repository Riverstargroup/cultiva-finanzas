# GPT Image 2 Asset Prompt Pack - Pixel Art Direction

## Base Style

Use this exact base style at the beginning of every prompt:

```text
Create original cozy pixel art for "Cultiva Finanzas", a Mexican financial education garden game. 128x128 sprite canvas for characters, crisp pixel edges, limited palette, readable silhouette, no anti-aliased painterly rendering, no 3D render, no smooth airbrush, no generic AI illustration look. Use clean clusters, intentional dithering, 1-2 px dark outline, warm botanical palette, transparent background, no text, no logos, no emojis, no copyrighted character references. Style should feel like a premium cozy mobile RPG with clear personality and simple animation-ready poses.
```

## Palette

Keep most assets inside this palette family:

- Forest outline: `#1B3B26`
- Leaf bright: `#4CAF50`
- Leaf muted: `#5B7A3A`
- Clay soft: `#D4C5B0`
- Warm cream: `#FEFBF6`
- Coin gold: `#E5B84B`
- Terracotta: `#B75E3C`
- Deep berry shadow: `#5D3136`

## Generation Rules

- Ask for **one sprite per image** when possible. Sheets are allowed only for concept exploration.
- Character sprite canvas: `128x128`.
- POI/world object canvas: `256x256`.
- Background tiles: `512x512`.
- Transparent background.
- Camera: mostly front or slight 3/4.
- Avoid gradients, painterly shading, fuzzy edges and soft shadows.
- Leave 8-12 px breathing room around the sprite.
- Use simple shadow blob only if needed.
